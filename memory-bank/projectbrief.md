# Project Brief

Smart Agenda AI is a React-based personal assistant application that combines a calendar/agenda view with an AI chat interface. It allows users to manage their schedule using natural language, supports recurring events, and integrates with Telegram for notifications and daily agendas.

## Core Features
- **AI Chat Interface**: Natural language processing to create, update, delete, and read events.
- **Calendar & Agenda Views**: "My Day" (Today) view and a general Calendar view.
- **Smart Reminders**: Browser notifications and Telegram alerts.
- **Telegram Integration**: Send daily agendas and event notifications to a Telegram bot.
- **Recurring Events**: Support for daily, weekly, monthly recurrence.
- **Offline First**: Uses IndexedDB (Dexie.js) for data persistence.
- **Theming**: Dark/Light mode support.

## Tech Stack
- React 19
- Vite
- TypeScript
- Tailwind CSS
- Dexie.js (IndexedDB)
- Lucide React (Icons)
- Google GenAI (AI processing)
