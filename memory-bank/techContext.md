# Tech Context

## Core Technologies
- **Frontend Framework**: React 19 (Hooks based).
- **Build Tool**: Vite.
- **Language**: TypeScript.
- **Styling**: Tailwind CSS (Utility-first).

## Data & State
- **Local Persistence**: Dexie.js (Wrapper for IndexedDB).
- **Local State**: React `useState`, `useEffect`.
- **Global Config**: `localStorage` used for Theme, AI Settings, Notified Events.

## External Services
- **Google GenAI**: Used for parsing user natural language inputs.
- **Telegram API**: Direct HTTP requests (likely) or via strict service for sending notifications.
